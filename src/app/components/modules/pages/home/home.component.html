POKE API

Este sitio web proporciona una API RESTful para objetos muy detallados creados a partir de miles de líneas de datos relacionados con Pokémon. Cubrimos específicamente la franquicia de videojuegos. Usando este sitio web, puedes consumir información sobre Pokémon, sus movimientos y mucho, mucho más.

¿Que es una API?

API (Application Programming Interface) Las API son mecanismos que permiten a dos componentes de software comunicarse entre sí mediante un conjunto de definiciones y protocolos.

Categoria De API

Api Pokémon se puede categorizar como una Api pública, la cual nos permite traer parámetros de búsqueda de información relacionada con Pokémon que pueden ser usados de manera libre.

Propósito

Este proyecto tiene como propósito mostrar las habilidades adquiridas durante el curso de capacitación, de manera que se optó por usar la API de Pokémon con fines educativos.
